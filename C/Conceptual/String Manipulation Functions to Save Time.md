## `<string.h>`
| Function Name | Input Args | Description |
|——————————————————————————————————————————|
|`strstr`|`char* s, char* t`|Returns index of first occurence of string `t` in string `s`|
|`strcmp`|``char* s, char* t`|Compares string `s` and string `t`|
|`strcpy`|`char* s, char* t`|Copies `t` to `s`|
